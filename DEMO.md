# Демо SelecGel для заказчика

## Быстрый старт (5 минут до презентации)

```bash
cd d:\vodopritok\app
python -m cli serve --port 8080
```

| Шаг | URL | Действие |
|-----|-----|----------|
| 1 | http://127.0.0.1:8080/screening | Ввести T, salinity заказчика → **Запустить** |
| 2 | Тот же экран | Показать stages 500→150→5 и top-5 |
| 3 | http://127.0.0.1:8080/pitch | Таблица «ТЗ → deliverables» для отклика |
| 4 | http://127.0.0.1:8080/reports | **Сгенерировать 6 .docx** → ZIP |

Данные screening **сохраняются** в `app/output/demo_session.json` и подставляются в отчёты.

## Демо-кейс по умолчанию

- Месторождение Западная Сибирь, Well-101  
- T=85°C, salinity 130 g/L, k=450 mD, WC=82%  
- Механизм: coning → Track 1 RPM  

Файл: `app/examples/reservoir_example.json`
