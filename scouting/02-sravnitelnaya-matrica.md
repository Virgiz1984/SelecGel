## 2. Селективность: количественные ориентиры из литературы

| Система | krw reduction | kro retention | Условия |
|---------|---------------|---------------|---------|
| Branched RPM (SPE 199334) | regain <20% | regain >75% | Sandstone, 93°C |
| HTHS gel (2026) | блокировка >95% | gas phase <35% block | 110°C, 160 g/L |
| HRPM field average | case-dependent | minimal oil loss | >3000 wells |
| PPG | channel PL reduction | oil bypasses | fracture-dominated |
| Катионный RPM (CNPC study) | Frrw >> Frro | Frr ↓ with perm↑ | Core flood |

## 3. Матрица «механизм обводнения → технология»

| Механизм | 1-й выбор | 2-й выбор | Не рекомендуется |
|----------|-----------|-----------|------------------|
| Water coning | RPM, weak gel | In-depth CDG | Bulk gel (без изоляции) |
| Channel behind casing | Cement + gel | — | RPM alone |
| Fracture channeling | RPPG, PPG | Bulk gel | RPM alone |
| High-perm streak | PPG, CDG | Bulk gel | RPM (thin streak) |
| Cross-flow layers | In-depth gel/PPG | RPM + profile control | Surface-active only |
| Bottom water gas well | HTHS selective gel | Foam | Standard RPM |

## 4. SWOT по приоритетным направлениям для собственной разработки

### A. HRPM / custom RPM

| Strengths | Weaknesses |
|-----------|------------|
| Простота закачки | Узкий window кандидатов |
| Минимальный skin | Слаб на трещинах |
| Большая field база | Конкуренция с Halliburton/DOSAS |

| Opportunities | Threats |
|---------------|---------|
| Локализация рецептуры | Патенты HRPM |
| Хемоинформатика мономеров | Неверный отбор скважин → репутационный риск |

### B. Термотропный полимер-неорганический гель (МЕГА-analog)

| Strengths | Weaknesses |
|-----------|------------|
| Российская НИР-база | TRL ниже коммерческих RPM |
| HTHS потенциал | Сложность масштабирования синтеза |
| «Интеллектуальное» поведение | Длительная оптимизация |

| Opportunities | Threats |
|---------------|---------|
| Уникальность для high-viscosity oils | Зависимость от минералогии керна |
| Совместимость с ТМС | Конкуренция с импортными гелями |

### C. RPPG / modified PPG

| Strengths | Weaknesses |
|-----------|------------|
| Доказанная эффективность в каналах | Не «фазовая» селективность |
| Контроль size/strength | Нужен fracture characterization |
| RPPG — next gen | Стоимость частиц |

| Opportunities | Threats |
|---------------|---------|
| Mature fields РФ | Washout при mismatch size |
| Hybrid с RPM | Большие объёмы закачки |

### D. Нанокомпозитный selective gel

| Strengths | Weaknesses |
|-----------|------------|
| HTHS + long-term stability | Высокая cost реагента |
| Высокая селективность gas/water | Мало field cases |
| Публикационная база 2025–2026 | Сложность QC нанофазы |
