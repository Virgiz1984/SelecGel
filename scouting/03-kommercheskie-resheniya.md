# Коммерческие решения и игроки рынка ОВП

## 1. Международные сервисные компании

### Halliburton

| Продукт/технология | Тип | Примечание |
|--------------------|-----|------------|
| HRPM (hydrophobically modified RPM) | Селективный RPM | >3000 wells, SPE 165091 |
| TOCP (Thixotropic Organically Crosslinked Polymer) | Gel conformance | SPE 231005, horizontal wells |
| Organically crosslinked polymer gels | Bulk gel | Mexico, North Sea field history |
| Thixotropic conformance sealant | Gel | SPE 186248, deviated wells |

**Позиционирование:** лидер по RPM + integrated diagnostics for candidate selection.

### Schlumberger

- Conformance / water shutoff portfolio (gel systems, particulate)
- Интеграция с reservoir evaluation и well intervention
- Конкретные trade names зависят от региона (WaterShut, gel treatments)

### Baker Hughes / legacy Nalco Champion

- Polymer gels, RPM-type treatments
- Water management chemicals for production optimization

### DOSAS B.V. (Нидерланды)

**Relative Permeability Modifier (RPM):**

- Polymer + internal crosslinker blend
- Salinity: high; Thermal: **до 165°C**
- Permeability: **до 5 D**
- Механизм: adsorption + capillary collapse → krw↓
- Также позиционируется для light sand control

### SNF / Kemira / другие химические производители

- Поставщики PAM, copolymers (AMPS, NVP), crosslinker packages
- База для **собственных рецептур** без полного сервисного цикла

## 3. Российский рынок

### Промысловые технологии (широкое применение, Западная Сибирь)

По обзору физико-химических методов ОВП на месторождениях ЗС (наибольший объём):

1. БЭС — бесполимерные эмульсионные составы  
2. ВДС — волокнисто-дисперсные  
3. ВУС / гл. ВУС — вязкоупругие  
4. ПАВ  
5. ВУС + ПАВ  
6. Полимер-дисперсные составы  
7. **ПГС** — полимер-гелевые системы (ГОС + кремнийорганический закрепитель)  
8. РПДН  
9. Жидкое стекло  
10. СС, ЭДС, ЭПС, ЭС  

**Характеристика:** преимущественно **неселективные или слабо селективные** блокирующие составы; высокая зрелость процессов, низкая стоимость, но ограниченный эффект при coning/channeling без селективности.

### Научные центры (потенциальные партнёры / бенчмарки)

| Организация | Направление |
|-------------|-------------|
| **ИХН СО РАН** (Томск) | Термотропные композиции МЕГА, «гель в геле», ограничение водопритока + EOR |
| **Институт нефтехимии СО РАН** | Полимерная химия для нефтедобычи |
| **ТИНГ НГУ / профильные кафедры** | Лабораторные испытания, core flood |
| **ВНИИнефть / Газпром нефть R&D** | Внутренние программы conformance (публично — полимер-дисперсные системы) |

### Сервисные подрядчики РФ

- Крупные НКК используют **смешанную модель**: собственные рецептуры + сервис бурения/ГТМ
- Импортные RPM/gel системы — **ограничения санкций и логистики** → аргумент в пользу **локальной разработки**

## 4. Карта «build vs buy»

```
                    Селективность
                         ↑
                         │
    HRPM (Halliburton)   │   Custom RPM + хемоинформатика ← ЦЕЛЕВАЯ ЗОНА
    DOSAS RPM            │   Термотропный МЕГА-type
                         │
    ─────────────────────┼─────────────────────→ Зрелость / готовность
                         │
    ПГС, ВУС, БЭС (РФ)   │   RPPG localized production
    Bulk gel (import)    │   Nanocomposite gel R&D
                         │
                         ↓
                    Низкая селективность
```

**Рекомендация для проекта:** не покупать «чёрный ящик» HRPM, а разработать **аналог RPM или термотропного геля** с патентным ландшафтом и адаптацией под конкретные пласты заказчика.

---

## 5. Патентный ландшафт (ориентиры для FTO)

При синтезе новых рецептур проверить:

- Hydrophobically modified polyacrylamides (HRPM patents, Halliburton)
- Organic crosslinked polymer gels (phenol-formaldehyde, PEI)
- RPPG composition and re-crosslinking chemistry (university + SNF filings)
- Thermotropic nanostructured gels (ИХН / Russian patents)
- Nano-SiO₂ reinforced polymer gels for HTHS (recent CN/International filings)

**Действие:** перед лабораторией — **FTO-скрининг** по мономерам, цепочке сшивки и nano-additives.

## 6. Критерии выбора vendor vs собственная разработка

| Критерий | Vendor (HRPM/PPG) | Собственная разработка |
|----------|-------------------|------------------------|
| Time to OPR | 1–3 мес. | 6–12 мес. |
| Адаптация к пласту | Ограничена | Полная |
| Стоимость лицензии/реагента | $$$ | CAPEX R&D, дешевле OPEX |
| IP ownership | Нет | Да |
| Санкционные риски | Есть | Ниже |
| Селективность под задачу | Generic | Targeted |
