## Ключевой вывод скаутинга

Наиболее перспективные **селективные** подходы для собственной разработки:

1. **RPM (Relative Permeability Modifier)** — гидрофобно-модифицированные полимеры, adsorption + swelling/shrinking
2. **Термотропные / «интеллектуальные» гели** — низкая вязкость при закачке, гелеобразование in situ (российская школа ИХН СО РАН)
3. **PPG / RPPG** — для трещин и высокопроницаемых каналов; RPPG — re-crosslinkable bulk gel
4. **Нанокомпозитные гели** — nano-SiO₂, полимер-наночастица для HTHS условий
5. **Хемоинформатика + DoE** — ускорение подбора мономеров, сшивателей и ПАВ под конкретную ПВ/минералогию

Механические методы (цементирование, пакеры, МУН) рассматриваются как дополнение, не как целевая «селективная химия».
